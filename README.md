# NYTNews-Scraper

News scraper  app that uses cheerio to scrape a news site, mongodb to save news articles and comments.

## Link
app: https://nytnews-scraper.herokuapp.com/

## Concepts and Technologies

* NPM => Package.Json
* Express
* Cheerio
* Node.js, Command line application
* Router
* Handlebars
* Heroku
* Require
* Mongoose
* Axios
* Margon
* Request
